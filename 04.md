# Анализ размера переменнных

Т.к. переменные известного размера могут принимать значения только из интервалов, поэтому анализ можно построить вокруг 
анализа интервалов, где каждый уровень в решетке содержит интервалы 
длины очередной степени двойки (т.к. они замкнуты по сложению и умножению).

Решетку необходимо за-lift-ить, тогда нижний top будет означать bigint, верхний - any

На практике имеет смысл выбрать какой-то widening для этой решетки, чтобы соблюсти бадланас медду производительнтсью и точностью. Возьмём набор из всех констант в cfg (базово) и добавим к нему степени двойки.

Также в анализе (как и просто анализе интервалов фигурирует лифтинг этой решетки).
