# Сломанный анализ 

Следующая программа сейчас ломает анализ с отслеживанием контекста:

```
if (x > 15) {
    open();
    flag = 1;
}

other_flag = flag;

if (other_flag) {
    close();
}
```

На данный момент описанные правила не умеют обрабатывать присваивание в флаги неконстантных выражений. 

Можно научиться обрабатывать нерекурсивные присваивания вида "переменная = пропозиционная формула" - 
опустошаем те контексты, где формула даёт 0, а переменная указана как 1 и наоборот.
